Caboodle MakeUp

React Web App

This customizable makeup app allows beauty influencers to share their favourite products recommendations. Users can sign in with Google authentication before being taken to an initial profile page. Searchable products from the MakeUp API can be added to the personal dashboard or the user's profile. All user information is stored in Firebase.

Made by Chelsea Roman, Gage Wagner, Eugenia Lau, Vinh Luu
